# Skill: CuraSync UI Improvement

## Role
You are a senior frontend engineer and UI/UX designer.

## Objective
Improve the UI/UX of the existing CuraSync application without changing backend logic, APIs, or data structures.

The system is already functional. Focus only on visual design, layout, and usability.

---

## Constraints
- Do NOT modify backend code
- Do NOT change API responses
- Do NOT break existing functionality
- Only improve frontend UI and UX

---

## Tasks

### 1. Dashboard UI
- Convert layout to card-based design
- Highlight key health metrics:
  - Blood Pressure (BP)
  - Heart Rate (HR)
  - Sugar
  - Risk level
- Show:
  - Value
  - Source (Manual / Alert / Report / Wearable)
  - Timestamp
- Use color coding:
  - Green (normal)
  - Yellow (moderate)
  - Red (high)

---

### 2. Monitoring Visualization
- Add line charts for:
  - BP trend
  - HR trend
- Display last 3–7 readings
- Show labels:
  - Rising
  - Stable
  - Fluctuating

---

### 3. Report Page
- Replace raw text with structured sections:
  - Vitals
  - Activity
  - Sleep
  - Risk Summary
- Use icons and spacing for clarity

---

### 4. Doctor Dashboard
- Convert patient list into cards
- Show:
  - Name
  - Risk level
  - Last update
- Add actions:
  - View Report
  - Disconnect Patient

---

### 5. Source Visibility
Replace:
"Synced from smartwatch"

With:
"Source: Manual / Alert / Report / Wearable"

Display clearly in UI.

---

### 6. Quick Add UI
- Create a “Quick Add Reading” component
- Fields:
  - BP
  - Sugar
  - Symptoms
- Keep minimal and fast

---

### 7. Design System
- Use consistent:
  - Colors
  - Typography
  - Spacing
- Add:
  - Rounded corners
  - Soft shadows
  - Clean padding

---

### 8. Responsiveness
- Ensure mobile-friendly UI
- Responsive layouts and components

---

### 9. UX Enhancements
- Add loading states (skeletons)
- Add empty states
- Add subtle animations (hover effects, transitions)

---

## Output
- Improved UI components
- Clean layouts
- Better visual hierarchy
- Fully functional frontend (no backend changes)

---

## Goal
Make CuraSync look like a professional healthcare product with modern UI and excellent user experience.
